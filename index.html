About ROTI Polar Map Viewer

This static app reads selected files locally in your browser. Files are not uploaded to a server.

The map uses corrected magnetic latitude and magnetic local time. ROTI shows TEC fluctuation activity, not absolute TEC.

Caveat: The detected boundary is a ROTI-based proxy of the high-latitude ionospheric irregularity belt. It should not be interpreted as an optical auroral oval boundary.

Source labels are placeholders for IGS/CDDIS and Cherniak ROTI product references until exact project URLs are configured.
